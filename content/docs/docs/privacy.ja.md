## はじめにおよび発効条項

**Open Endfield Map**（以下「当社」、「本プロジェクト」、または「OEM」といいます）へようこそ。当社は、お客様の個人情報のプライバシーとデータセキュリティの重要性を深く認識しており、最高水準でお客様の個人情報を保護することに尽力しています。本『プライバシーポリシー』は、お客様が本プロジェクトのサービスにアクセスして利用する際に、当社がどのようにお客様の個人データを収集、使用、保存、共有、および保護するかを明確かつ透明に説明するとともに、これらのデータに対してお客様が有する管理権を明確にすることを目的としています。

**本ポリシーを注意深くお読みください。本プロジェクトのサービスにアクセスまたは利用した場合、お客様は本プライバシーポリシーに記載されているデータ処理の運用を読み、理解し、同意したものとみなされます。**

> **注意**：本プロジェクトは、「ローカルファースト（Local-First）」の設計を採用しています。デフォルトの設定および未登録/未ログインの状態では、お客様のすべての利用データは**お客様のローカルブラウザ内にのみ保存**され、当社がコアなインタラクションデータを自発的に収集、アップロード、または分析することはありません。

## 当社が収集する情報および収集方法

安全で安定した地図サービスおよびインタラクティブ機能を提供するため、当社は必要最小限の範囲内で以下の種類の情報のみを収集します。

### お客様が自発的に提供する情報

本プロジェクトの特定の高度な機能（コンテンツの貢献、アカウントの同期など）を利用することを選択した場合、以下の情報の提供をお願いすることがあります。
* **アカウント情報**：アカウントを登録する際、ユーザー名、メールアドレス、および暗号化処理されたパスワードの認証情報を収集します。
* **ユーザー生成コンテンツ（UGC）**：お客様が自発的に地図上にポイントのマーキング、修正の提案、編集履歴の送信、またはコメントの投稿を行った場合、これらのコンテンツデータおよび関連するアカウント識別子を収集します。
* **コミュニケーション記録**：サポートの要請や問題の報告のために、メールまたはGitHub Issueを通じて当社に連絡した場合、関連する通信記録を保存します。

### 当社が自動的に収集するテレメトリおよび運用データ

お客様が本プロジェクトのウェブサイトにアクセスされる際、当社のサーバーおよびセキュリティ保護機構は、サービスの可用性および安全性を確保するため、一定の技術的データを自動的に記録します。

* **ネットワークログデータ**：お客様の一部または全部のIPアドレス、ブラウザの種類およびバージョン、オペレーティングシステム、参照元URL、閲覧されたページのパス、ならびにタイムスタンプを含みます。
* **デバイスおよびセッション情報**：ログイン状態の維持、言語設定の保持、および基本的なパフォーマンス監視のための識別子。
* **コア進捗データ**：マップの閲覧状況やマーカーの状態を含み、複数のデバイス／環境間で進捗を同期する目的で使用されます。

### ローカル保存データ（当社が収集しない項目）

* **ローカルデータベース**：お客様のカスタム設定、フィルター等の非コアデータは、原則としてお客様の端末内の `LocalStorage` または `IndexedDB` に保存されます。**これらのデータはお客様が完全に管理するものであり、お客様が自らクラウド同期機能を有効化しない限り、当社が取得することはありません。**

## サードパーティサービスおよびデータの流通

効率的で安全なグローバルサービスを提供するため、本プロジェクトは一部のサードパーティ・インフラストラクチャおよびAPIサービスを統合しています。これらのサービスは、データ処理者（Data Processors）としてお客様の少量のデータを処理する場合があり、それぞれのプライバシーポリシーの対象となります。

| サードパーティサービス提供者 | サービスの用途およびデータインタラクションの説明 |
| :--- | :--- |
| **Cloudflare** | **用途**：グローバルCDNアクセラレーション、DDoS防御、DNS解決。<br>**処理するデータ**：IPアドレス、基本的なネットワークリクエストの特徴。<br>**説明**：悪意のある攻撃からウェブサイトを保護し、コンテンツの配信を高速化するために使用されます。 |
| **OpenAI (Moderation API)** | **用途**：ユーザー生成コンテンツの自動化されたコンプライアンスおよびセキュリティ審査。<br>**処理するデータ**：お客様が送信した公開テキストコンテンツ（コメント、マーカーの説明など）。<br>**説明**：違反/悪意のある情報が含まれているかどうかを識別するためにのみ使用されます。OpenAI APIのポリシーに基づき、このようなデータが彼らのAIモデルのトレーニングに使用されることは**ありません**。 |
| **Upstash** | **用途**：サーバーレスデータベースキャッシュおよびAPIリクエストのレート制限。<br>**処理するデータ**：ハッシュ化されたIP識別子、一時的なセッショントークン。<br>**説明**：インターフェースの悪用を防ぎ、バックエンドサービスの安定性を保障するために使用されます。 |

## お客様の情報の使用方法

当社は、以下の正当な目的および法的根拠に基づいて、お客様の個人データを処理します。

1. **サービスの提供および維持**（*契約の履行*）：クロスデバイスのデータ同期や地図編集の表示などのコア機能を実現するため、お客様のアカウントおよびコンテンツデータを処理します。
2. **セキュリティおよびコンテンツモデレーション**（*正当な利益*）：自動化ツール（OpenAI Moderationなど）や手動の手段を利用してコンテンツを審査し、スパム、悪意のあるスクリプト、または違法コンテンツの拡散を防ぎます。
3. **パフォーマンスの最適化および分析**（*正当な利益*）：匿名化されたログデータを分析してユーザーのアクセストレンドを把握し、地図の読み込み速度やシステムアーキテクチャを最適化します。
4. **サービス通知**（*契約の履行*）：パスワードリセットのメールやアカウント状態の変更通知など、必要なトランザクション関連またはセキュリティ関連の通知をお客様のメールアドレスに送信します。

## データの保存、セキュリティ、および保持期間

### データセキュリティ保護措置

当社は、お客様のデータを保護するため、業界標準に準拠した物理的、技術的、および管理上の措置を講じています。これには以下が含まれますが、これらに限定されません。
* すべてのデータ転送は、最新のTLS/HTTPS暗号化プロトコルの使用を強制されます。
* パスワードは強力なハッシュアルゴリズム（bcrypt/Argon2など）を使用して一方向の暗号化保存が行われ、いかなる者（プロジェクトのメンテナーを含みます）もお客様の平文のパスワードを取得することはできません。
* 厳格なデータベースへのアクセス権限管理。

### データの保持期間

* **アカウントデータ**：お客様のアカウントがアクティブな状態である限り、登録情報およびクラウド同期データを保持し続けます。
* **ログおよびテレメトリデータ**：サーバーのアクセスログおよびセキュリティブロックログは、通常、生成から30日以内に自動的にローリング上書きされるか、または永久に削除されます。
* **コンテンツの貢献**：Wiki/クラウドソーシングの性質を持つ地図プロジェクトであるため、お客様が送信した公開の変更やマーキングの記録は、地図の進化の履歴バージョンを保存するため、プロジェクトと共に存続する場合があります。お客様がアカウントを退会した場合でも、その貢献コンテンツは匿名またはアーカイブ形式で保持される可能性があります。

## Cookieおよび類似技術

本プロジェクトはCookieの乱用を厳格に制限し、必要なシナリオにおいてのみこれらの技術を使用します。

* **絶対に不可欠なCookie**：お客様のログインセッション（Session）を維持し、クロスサイトリクエストフォージェリ（CSRF）攻撃を防ぐために使用されます。
* **機能的なローカルストレージ**：当社は、ダークモードの好み、地図ビューのズームレベル、およびローカルマーカーを保存するために、HTML5 Web Storage（`LocalStorage` / `IndexedDB`）を優先して使用します。
* **管理の選択**：お客様は、ブラウザの設定を通じていつでもCookieおよびローカルストレージをクリアまたは無効にすることができます。これらの機能を無効にした場合、設定の再構成が必要になるか、またはアカウント同期などの機能が使用できなくなる可能性があることにご注意ください。詳細については、[『Cookieポリシー』](./cookie)をご参照ください。

## データの共有および開示

当社は、お客様の個人データをいかなる第三者にも販売または貸与**しないことをお約束します**。当社は、以下の極めて限定的な状況においてのみデータを共有します。

1. **サービス提供者へのコンプライアンス**：「最小限の必要性の原則」に基づき、本ポリシー第3節で言及された信頼できるサードパーティサービスプロバイダーと、サービスの運用を維持するために必要なデータを共有します。
2. **法的コンプライアンスおよび強制的な要求**：適用される法律、規制、有効な召喚状、裁判所の命令、またはその他の法的手続きを遵守するために必要であると当社が確信する場合。
3. **正当な権利の保護**：潜在的なセキュリティの脅威、詐欺行為、本プロジェクトのシステムに対する悪意のある攻撃を調査、防止、または対処するため、あるいは本プロジェクトおよび公衆の財産と生命の安全を保護するため。

## お客様のデータに関する権利

本項の内容は、GDPRやCCPAなどの関連法規に適用されます。お住まいの司法管轄区に応じて、お客様はご自身の個人データに対して広範な管理権を有します。当社のデータベースおよびRedisキャッシュレイヤーに保存されているデータに関して、お客様は以下の権利を有します。

* **アクセス権およびデータポータビリティ権**：当社が保持するお客様に関する個人データのコピーの取得を要求する権利。
* **訂正権**：データが不正確または不完全である場合、当社に修正を要求する権利。
* **削除権（忘れられる権利）**：アカウントおよび関連する個人を特定できる情報の削除をいつでも申請する権利。
* **同意の撤回権**：お客様の同意に基づいて行われるデータ処理活動について、いつでもその同意を撤回する権利。

**権利の行使方法**：
上記の権利を行使するには、`privacy@opendfieldmap.org` 宛てに電子メールを送信するか、GitHubのオープンソースリポジトリでIssueを提出してください。ご請求を受領してから180営業日以内に対応いたします。お客様のローカルブラウザ内に完全に保存されているデータについては、お客様ご自身でブラウザのキャッシュをクリアすることで削除できます。

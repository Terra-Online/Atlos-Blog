## 定義と背景

- **本プロジェクト**：開発チーム（[OEM-1](https://github.com/orgs/Terra-Online/teams/oem-1)内の[cirisus](https://github.com/cirisus)、[mwx0621](https://github.com/mwx0621)、[502y](https://github.com/502y)等の貢献者を含みます）によって開発・保守されている[終末地地図（Open Endfield Map）](https://opendfieldmap.org/)のウェブサイトおよび関連サービスを指します。ゲーム内のデータに基づいた地図の閲覧、マーカーの分析、進捗状況の追跡、クラウド同期などの機能をユーザーに提供することを目的としています。
- **サービス**：本プロジェクトが提供する、地図の表示、ルート計画、ユーザーインタラクション、アカウントシステムサービス、およびデータの保存・アクセス、進捗の追跡などの機能を指します。
- **ユーザー**：本プロジェクトにアクセス、および/または利用する個人または組織を指します。
- **ユーザーアカウント**：ユーザーが本プロジェクトの高度な機能（クラウドデータ同期、クロスデバイスアクセスなど）を利用するために登録し所有する、専用のデジタル認証情報を指します。

> **注意：** 本プロジェクトの詳細なプライバシーデータ収集基準、Cookieの使用説明、API利用規範、およびコンテンツの出典については、別途定める[『プライバシーポリシー』](./privacy)、[『Cookieポリシー』](./cookie)、[『API利用規約』](./api-terms)、および[『コンテンツの出典』](./credits)をご参照ください。本利用規約に同意することにより、上記の付属ポリシーについても承諾し同意したものとみなされます。

## データの保存とプライバシーの保護

データの保存戦略は、ユーザーの利用状態に依存します。

### 未ログイン状態における保存状況

- アカウントにログインしていないユーザーの個人的な設定、マーカーの進捗、フィルターの状態などのデータは、**ユーザーが現在使用しているブラウザのローカル（Local Storage / IndexedDB）に完全に保存されます**。
- このモードでは、デバイスの切り替え、ブラウザの変更、シークレットモードの有効化を行うと、**データは引き継がれません**。ゲストユーザーには、定期的に「データのエクスポート/インポート（Export/Import Data）」機能を使用してローカルバックアップを行うことを推奨します。

### ログイン状態における保存状況

- 登録してアカウントにログインしたユーザーの進捗データおよびアプリケーション設定は、クロスデバイスの互換性を実現するために、**暗号化されて当社のD1データベースに同期・保存されます**。
- 開発チームは、収集したデータ（アカウント識別子、暗号化されたパスワード、暗号化されたゲーム進捗データなど）が**地図の同期サービスを提供するためにのみ使用される**ことをお約束します。
- 本プロジェクトは、お客様の個人データおよび操作履歴を第三者の営利機関に販売、共有、または漏洩することは**ありません**。

### データセキュリティと免責

- 開発チームは、データベースのセキュリティを保護するために業界標準の暗号化措置を採用していますが、本プロジェクトが非営利目的のオープンソースプロジェクトであることに鑑み、エンタープライズレベルの絶対的なセキュリティを保証することはできません。
- **不可抗力、悪意のあるハッカー攻撃、またはサードパーティサービス（クラウドプロバイダーなど）の障害に起因するユーザーデータの漏洩や喪失について、本プロジェクトおよび開発チームは法的および経済的な賠償責任を免責されるものとします。**

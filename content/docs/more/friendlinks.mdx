Open Endfield Map is made possible by the support of exceptional infrastructure providers and a thriving open community. We are grateful to each of these partners.

## Infrastructure Partners

- **[Cloudflare](https://www.cloudflare.com/)** — Global CDN, DDoS mitigation, DNS, and Pages hosting that keep Open Endfield Map fast and resilient for every user on earth.
- **[OpenAI](https://openai.com/)** — The Moderation API that automatically screens user-generated content, keeping our community safe and policy-compliant.
- **[Upstash](https://upstash.com/)** — Serverless Redis and Kafka for low-latency caching, rate limiting, and event streaming — built for edge-first architectures.

## Community Friends

- **[Next.js](https://nextjs.org/)** — The React framework powering this documentation site with full-stack capabilities.
- **[Fumadocs](https://fumadocs.vercel.app/)** — The documentation framework that provides the elegant sidebar, search, and multi-language support you see here.

## How to Contribute

We welcome contributions from everyone! Here are some ways you can help:

### Code Contributions

1. Fork the repository on GitHub
2. Create a feature branch
3. Submit a pull request

### Map Data

Help us by submitting map annotations, correcting existing data, or reporting inaccuracies.

### Documentation

Improve our documentation by fixing typos, adding examples, or translating content.

### Community

- Join discussions in GitHub Issues
- Help answer questions from other users
- Share the project with fellow players

## Code of Conduct

All contributors are expected to follow our Code of Conduct. Be respectful, inclusive, and constructive.

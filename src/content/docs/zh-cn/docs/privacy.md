## 引言与生效条款

欢迎使用 **Open Endfield Map**（以下简称“我们”、“本项目”或“OEM”）。我们深知个人隐私及数据安全对您的重要性，并致力于以最高标准保护您的个人信息。本《隐私权政策》旨在清晰、透明地向您说明在您访问和使用本项目服务时，我们如何收集、使用、存储、共享以及保护您的个人数据，同时明确您对这些数据所拥有的管理权利。

**请仔细阅读本政策。当您访问或使用本项目的服务时，即表示您已阅读、理解并同意本隐私政策中描述的数据处理操作。**

> **注意**：本项目采用“本地优先（Local-First）”的设计。在默认且未注册/登录的状态下，您的所有使用数据均**仅保存在您的本地浏览器中**，我们不会主动收集、上传或分析核心交互数据。

## 我们收集的信息及收集方式

为了向您提供安全、稳定的地图服务及互动功能，我们仅在必要范围内收集以下类型的信息：

### 您主动提供的信息

当您选择使用本项目的特定高级功能（如内容贡献、账户同步）时，我们可能需要您提供以下信息：
* **账户信息**：当您注册账户时，我们会收集您的用户名、电子邮箱地址以及经过加密处理的密码凭证。
* **用户生成内容（UGC）**：当您主动在地图上提交点位标注、纠错建议、编辑记录或发表评论时，我们会收集上述内容数据以及相关的账户标识。
* **沟通记录**：当您通过电子邮件或 GitHub Issue 与我们联系以获取支持或反馈问题时，我们将保存相关通信记录。

### 我们自动收集的遥测与运行数据

在您访问本项目网站时，我们的服务器和安全防护机制会自动记录部分技术数据，以确保服务的可用性与安全性：
* **网络日志数据**：包括您的部分或全部 IP 地址、浏览器类型与版本、操作系统、来源/引荐 URL、访问的页面路径及时间戳。
* **设备与会话信息**：用于维持登录状态、语言偏好和基础性能监控的标识符。
* **核心进度**：包括您的地图浏览进度、标记状态，用于维持您的进度跨设备/环境同步。

### 本地存储数据（非收集项）

* **本地数据库**：您的自定义偏好、筛选器等非核心数据，默认存储于您设备的 `LocalStorage` 或 `IndexedDB` 中。**这部分数据由您完全控制，除非您主动触发云端同步功能，否则我们无法获取。**

## 第三方服务与数据流转

为了提供高效且安全的全球化服务，本项目集成了部分第三方基础设施与 API 服务。这些服务可能作为数据处理者（Data Processors）处理您的少量数据，并受其各自隐私政策的约束：

| 第三方服务提供商 | 服务用途及数据交互说明 |
| :--- | :--- |
| **Cloudflare** | **用途**：全球 CDN 加速、DDoS 防护、DNS 解析。<br>**处理数据**：IP 地址、基础网络请求特征。<br>**说明**：用于保护网站免受恶意攻击及加速内容分发。 |
| **OpenAI (Moderation API)** | **用途**：用户生成内容的自动化合规与安全审核。<br>**处理数据**：您提交的公开文本内容（如评论、标注描述）。<br>**说明**：仅用于鉴别是否包含违规/恶意信息。根据 OpenAI API 政策，此类数据**不会**被用于其 AI 模型的训练。 |
| **Upstash** | **用途**：无服务器数据库缓存与 API 请求限流。<br>**处理数据**：经过哈希处理的 IP 标识符、临时会话令牌。<br>**说明**：用于防止接口滥用，保障后端服务的稳定性。 |

## 我们如何使用您的信息

我们基于以下合法目的及法律依据处理您的个人数据：

1. **提供与维护服务**（*履行合同*）：处理您的账户和内容数据，以实现数据跨设备同步、地图编辑展示等核心功能。
2. **安全与内容审核**（*合法利益*）：利用自动化工具（如 OpenAI Moderation）和人工手段审核内容，防止垃圾信息、恶意脚本或违法内容的传播。
3. **性能优化与分析**（*合法利益*）：分析去标识化的日志数据，了解用户访问趋势，从而优化地图加载速度和系统架构。
4. **服务通知**（*履行合同*）：向您的邮箱发送必要的交易类或安全类通知，如密码重置邮件、账户状态变更提醒。

## 数据的存储、安全与保留期限

### 数据安全保护措施
我们实施了符合行业标准的物理、技术和管理措施来保护您的数据。包括但不限于：
* 所有数据传输均强制使用现代 TLS/HTTPS 加密协议。
* 密码使用强哈希算法（如 bcrypt/Argon2）进行单向加密存储，任何人（包括项目维护者）均无法获取您的明文密码。
* 严格的数据库访问权限控制。

### 数据保留期限
* **账户数据**：只要您的账户处于活动状态，我们将一直保留您的注册信息和云端同步数据。
* **日志与遥测数据**：服务器访问日志和安全拦截日志通常在生成后的 30 天内被自动滚动覆盖或永久删除。
* **内容贡献**：作为维基/众包性质的地图项目，您提交的公开修改或标注记录可能会与项目共存，以保留地图演进的历史版本。即使您注销账户，该贡献内容也可能以匿名或归档形式保留。

## Cookie 及同类技术

本项目严格限制 Cookie 的滥用，仅在必要场景下使用此类技术：

* **绝对必要的 Cookie**：用于维持您的登录会话（Session）和防止跨站请求伪造（CSRF）攻击。
* **功能性本地存储**：我们优先使用 HTML5 Web Storage (`LocalStorage` / `IndexedDB`) 来保存您的深色模式偏好、地图视图层级及本地标记。
* **管理选择**：您可以随时通过浏览器设置清除或禁用 Cookie 及本地存储。请注意，禁用这些功能可能会导致您需重新配置偏好，或无法使用账户同步等功能。

## 数据共享与披露

我们承诺**绝不向任何第三方出售或出租您的个人数据**。我们仅在以下极其有限的情况下共享数据：

1. **依从服务提供商**：基于“最小必要原则”，与本政策第三节提及的受信任第三方服务商共享维持服务运行所需的数据。
2. **法律合规与强制要求**：当我们确信为了遵守适用的法律、法规、有效传票、法院命令或其他法律程序时。
3. **保护合法权益**：为了调查、防止或处理潜在的安全威胁、欺诈行为、对本项目系统的恶意攻击，或保护本项目及公众的财产和生命安全时。

## 您的数据权利

本款内容适用于 GDPR、CCPA 等相关法规；根据您所在的司法管辖区，您对自己的个人数据享有广泛的控制权。针对存储在我们数据库与 Redis 缓存层上的数据，您拥有以下权利：

* **访问与导出权**：您有权要求获取我们所持有的关于您的个人数据副本。
* **更正权**：如果您的数据不准确或不完整，您有权要求我们进行修正。
* **删除权（被遗忘权）**：您可以随时申请删除您的账户及关联的个人可识别信息。
* **撤回同意权**：对于基于您同意而进行的数据处理活动，您有权随时撤回该同意。

**如何行使权利**：
您可以通过向 `privacy@opendfieldmap.org` 发送电子邮件，或在我们的 GitHub 开源仓库中提交 Issue 来行使上述权利。我们将在收到请求后的 180 个工作日内予以响应。对于完全存储在您本地浏览器中的数据，您可以自行通过清除浏览器缓存进行删除。

## 一、定义与背景

- **本项目**：指由开发者团队（包括 OEM-Team-1 中的 @cirisus, @mwx0621, @502y 等贡献者）开发和维护的“终末地地图集（Open Endfield Map）”网站及相关服务，旨在为用户提供基于游戏内数据的地图查看、点位分析、进度跟踪及云端同步等功能。
- **服务**：指本项目所提供的地图展示、路径规划、用户交互、账号系统服务及数据存取与进度跟踪等功能。
- **用户**：指访问和/或使用本项目的任何个人或组织。
- **用户账号**：指用户为使用本项目的高级功能（如云端数据同步、跨设备访问等）而注册并拥有的专属数字凭证。

> **注意：** 本项目的详细隐私数据收集标准、Cookie 使用说明及 API 使用规范，请参见单独的用户隐私政策、Cookie 政策和 API 使用条款。接受本服务条款即视为您同时知晓并同意上述附属政策。

## 四、数据存储与隐私保护

数据存储策略依赖于用户使用状态。

### 4.1 未登录情形下的存储情况

- 未登录账号的用户，其个人偏好设置、标点进度、筛选器状态等数据，**完全保存在用户当前使用的浏览器本地（Local Storage / IndexedDB）**。
- 在此模式下，切换设备、更换浏览器、开启无痕模式**将不会继承您的数据**。我们建议游客用户定期使用“导出/导入数据（Export/Import Data）”功能进行本地备份。

### 4.2 未登录情形下的存储情况

- 注册并登录账号的用户，其进度数据和应用设置将**加密同步并存储至我们的 D1 数据库中**，以实现跨设备互通。
- 开发团队承诺，收集的数据（如账号标识符、加密后的密码及加密后的游戏进度数据）**仅用于提供地图同步服务**。
- 本项目**不会**将您的个人数据及操作记录出售、共享或泄露给任何第三方商业机构。

### 4.3 数据安全与免责

- 开发团队采用了行业标准的加密措施来保护数据库安全，但鉴于本项目为非盈利性质的开源项目，我们无法提供企业级的绝对安全承诺。
- **因不可抗力、恶意黑客攻击、或第三方服务（如云服务商）故障导致的用户数据泄露或丢失，本项目及开发团队免于承担法律及经济赔偿责任。**
- 用户拥有对自己账号数据的完全控制权。您可以随时在设置中选择“删除账号”，我们将从服务器端彻底擦除您的所有数据（此操作不可逆）。

## 帳戶與存取憑證
1. **憑證安全**：倘若 OEM 為閣下分配了 API 金鑰（API Key）或存取權杖（Access Token），閣下必須嚴格保密。閣下不得與任何第三方共用、出售、轉讓或以其他方式洩露閣下之 API 憑證。

## 允許之用途
閣下可出於個人、學習及非商業目的存取 OEM API。我們鼓勵開發者在 OEM 現有數據之基礎上，構建能提供實質創新體驗之衍生工具（例如：遊戲路線規劃工具、資源計算器、數據分析工具、Discord/QQ 機械人等）。閣下之應用程式必須為終端用戶提供 OEM 官方平台以外的**附加價值**。

## 禁止之用途
閣下於使用 OEM API 時，嚴禁進行以下行為：
1. **任何形式之商業化**：閣下不得利用 OEM API 或其提供之數據產生任何直接或間接之收入。此包括但不限於：於應用程式內置入廣告、設置付費訂閱（Paywalls）、強制性贊助/打賞，或將應用程式作為付費軟件出售。如需商業授權，必須事先取得 OEM 團隊之書面許可。
2. **鏡像、搬運或替代品開發**：閣下不得使用本 API「按原樣」複製、展示或重新分發 OEM 之地圖內容與互動介面。嚴禁利用本 API 開發旨在替代、分流 OEM 官方網站流量之套殼網站或複製版。
3. **批量抓取與數據集轉售**：閣下不得大規模下載、抓取、提取本 API 之基礎數據（如全量坐標點位、圖鑑文本），並將其作為閣下獨立之離線數據集，或透過閣下自身之 API 進行二次發布。
4. **濫用與破壞**：閣下不得以任何可能破壞、導致超載或損害 OEM 伺服器及網絡之方式使用本 API（包括但不限於 DDoS 攻擊、利用漏洞注入惡意程式碼等）。

## 存取限制與快取策略
1. **遵守頻率限制**：閣下必須遵守 OEM 於 API 回應標頭（Headers）或官方文件中規定之呼叫頻率限制（Rate Limits）。嚴禁透過多 IP 輪換、偽造 User-Agent 等手段規避限制。
2. **合理快取**：為保護我們之伺服器資源，閣下應於應用程式中實施合理之本機快取（Caching）策略。請避免對靜態或極少變更之數據進行高頻重複請求。

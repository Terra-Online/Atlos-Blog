# API 使用條款

本文件適用於所有存取或調用 Open Endfield Map API 之人士或機構。

---

## 一、API 性質

- 僅用於支援地圖功能  
- 不保證長期穩定或版本兼容  
- 可隨時修改或終止  

---

## 二、禁止行為

嚴禁：

1. 批量抓取地圖切片  
2. 高頻請求造成負載壓力  
3. 繞過限流機制  
4. 商業再分發  
5. 用於作弊或外掛工具  

---

## 三、限流機制

我們可能：

- 基於 IP 實施速率限制  
- 封鎖異常流量  
- 採用快取與識別機制防止濫用  

違規可能導致帳戶封禁或永久禁止存取。

---

## 四、資料權屬

- 僅限個人非商業用途  
- 遊戲資源版權屬官方所有  
- 開源代碼受 AGPL v3.0 約束  

---

## 五、免責聲明

API 按「現狀」提供，不作任何明示或默示保證。

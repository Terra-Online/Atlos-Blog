## 一、引言與生效條款

歡迎使用 **Open Endfield Map**（以下簡稱「我們」或「本項目」）。我們深明個人私隱及數據安全對閣下之重要性，並致力以最高標準保障閣下之個人資料。本《私隱政策》旨在清晰及透明地說明，當閣下存取及使用本項目服務時，我們如何收集、使用、儲存、共享及保護閣下之個人數據，以及閣下就該等數據所享有之管理權利。

**請閣下仔細閱讀本政策。當閣下存取或使用本項目之服務，即表示已閱讀、理解並同意本私隱政策所述之數據處理安排。**

> **注意**：本項目採用「本地優先（Local-First）」之設計。在預設及未註冊／未登入之情況下，閣下所有使用數據均**僅儲存在閣下之本地瀏覽器中**，我們不會主動收集、上載或分析核心互動數據。

## 二、我們收集之資料及收集方式

為向閣下提供安全及穩定之地圖服務與互動功能，我們僅於必要範圍內收集以下類型之資料：

### 2.1 閣下主動提供之資料

當閣下選擇使用本項目之特定進階功能（如內容貢獻、帳戶同步）時，我們可能需要閣下提供以下資料：

* **帳戶資料**：當閣下註冊帳戶時，我們將收集閣下之使用者名稱、電郵地址，以及經加密處理之密碼憑證。
* **用戶生成內容（UGC）**：當閣下於地圖上提交點位標註、糾錯建議、編輯記錄或發表評論時，我們將收集相關內容數據及其帳戶識別資料。
* **通訊記錄**：當閣下透過電郵或 GitHub Issue 聯絡我們以獲取支援或提交意見時，我們將保存相關通訊內容。

### 2.2 我們自動收集之遙測及運行資料

當閣下存取本項目網站時，我們之伺服器及安全防護機制會自動記錄部分技術資料，以確保服務可用性與系統安全：

* **網絡日誌資料**：包括閣下之部分或全部 IP 位址、瀏覽器類型及版本、操作系統、來源／引薦網址、所瀏覽頁面路徑及時間戳記。
* **裝置與會話資訊**：用於維持登入狀態、語言偏好及基礎效能監測之識別符。

### 2.3 本地儲存資料（非收集項目）

* **本地資料庫**：閣下之地圖瀏覽進度、標記狀態、自訂篩選偏好等核心資料，預設儲存於閣下裝置之 `LocalStorage` 或 `IndexedDB`。**該等資料完全由閣下控制；除非閣下主動啟用雲端同步功能，否則我們無法存取。**

## 三、第三方服務與數據流轉

為提供高效及安全之全球化服務，本項目整合部分第三方基礎設施及 API 服務。該等服務可能作為數據處理者（Data Processors）處理少量資料，並受其各自私隱政策規範：

| 第三方服務提供者 | 服務用途及資料交互說明 |
| :--- | :--- |
| **Cloudflare** | **用途**：全球 CDN 加速、DDoS 防護、DNS 解析。<br>**處理資料**：IP 位址及基本網絡請求特徵。<br>**說明**：用於保障網站免受惡意攻擊及加速內容分發。 |
| **OpenAI（Moderation API）** | **用途**：用戶生成內容之自動合規及安全審核。<br>**處理資料**：閣下提交之公開文本內容（如評論、標註描述）。<br>**說明**：僅用於識別是否包含違規或惡意資訊。根據 OpenAI API 政策，該等資料**不會**用於其人工智能模型訓練。 |
| **Upstash** | **用途**：無伺服器資料庫快取及 API 請求限流。<br>**處理資料**：經哈希處理之 IP 識別符及臨時會話權杖。<br>**說明**：用於防止介面濫用及保障後端服務穩定性。 |

## 四、我們如何使用閣下之資料

我們基於以下合法目的及法律依據處理閣下之個人資料：

1. **提供及維護服務**（*履行合約*）：處理閣下之帳戶及內容資料，以實現跨裝置同步及地圖編輯展示等核心功能。
2. **安全與內容審核**（*合法利益*）：利用自動化工具（如 OpenAI Moderation）及人工方式審核內容，以防止垃圾訊息、惡意程式碼或違法內容之傳播。
3. **效能優化與分析**（*合法利益*）：分析去識別化之日誌資料，以了解使用趨勢並優化地圖載入速度與系統架構。
4. **服務通知**（*履行合約*）：向閣下之電郵地址發送必要之交易或安全通知，例如重設密碼郵件及帳戶狀態變更提示。

## 五、資料儲存、安全與保留期限

### 5.1 資料安全措施

我們已採取符合行業標準之實體、技術及管理措施保障閣下之資料，包括但不限於：

* 所有資料傳輸均強制採用現代 TLS／HTTPS 加密協議；
* 密碼以強哈希演算法（如 bcrypt／Argon2）作單向加密儲存，任何人士（包括項目維護者）均無法取得明文密碼；
* 嚴格之資料庫存取權限控制機制。

### 5.2 資料保留期限

* **帳戶資料**：只要閣下帳戶維持有效狀態，我們將保留閣下之註冊資料及雲端同步數據。
* **日誌與遙測資料**：伺服器存取日誌及安全攔截日誌一般於生成後 30 日內自動輪替覆蓋或永久刪除。
* **內容貢獻**：作為具維基／眾包性質之地圖項目，閣下提交之公開修改或標註記錄可能與項目歷史版本一併保留。即使閣下註銷帳戶，有關內容亦可能以匿名或歸檔形式保存。

## 六、Cookie 及同類技術

本項目嚴格限制 Cookie 之使用，僅於必要情況下採用：

* **必要性 Cookie**：用於維持登入會話（Session）及防止跨站請求偽造（CSRF）攻擊。
* **功能性本地儲存**：我們優先使用 HTML5 Web Storage（`LocalStorage`／`IndexedDB`）保存深色模式偏好、地圖視圖層級及本地標記資料。
* **管理選項**：閣下可隨時透過瀏覽器設定清除或停用 Cookie 及本地儲存。請注意，停用後可能需重新設定偏好，或無法使用帳戶同步等功能。

## 七、資料共享與披露

我們承諾**絕不出售或出租閣下之個人資料予任何第三方**。僅於以下極有限情況下共享資料：

1. **服務供應商依從**：根據「最小必要原則」，向本政策第三節所述之受信任第三方服務供應商提供維持服務所需資料；
2. **法律合規要求**：當我們合理相信為遵守適用法律、法規、有效傳票、法院命令或其他法律程序所必須；
3. **保障合法權益**：為調查、防止或處理潛在安全威脅、欺詐行為、對本項目系統之惡意攻擊，或保障本項目及公眾之財產與人身安全。

## 八、閣下之資料權利

本條款適用於 GDPR、CCPA 等相關法規；視乎閣下所屬司法管轄區，閣下對其個人資料享有廣泛控制權。就儲存在我們資料庫及 Redis 快取層之資料，閣下享有以下權利：

* **查閱與匯出權**：有權要求取得我們所持有關於閣下之個人資料副本；
* **更正權**：如資料不準確或不完整，有權要求修正；
* **刪除權（被遺忘權）**：可隨時申請刪除帳戶及相關可識別個人資料；
* **撤回同意權**：對於基於閣下同意進行之數據處理活動，有權隨時撤回同意。

**行使權利方式：**  
閣下可透過發送電郵至 `privacy@opendfieldmap.org`，或於我們之 GitHub 開源倉庫提交 Issue 行使上述權利。我們將於收到請求後 180 個工作日內作出回應。至於完全儲存在閣下本地瀏覽器之資料，閣下可自行透過清除瀏覽器快取刪除。

隨著項目功能更新或法律法規變動，我們可能不時修訂本《私隱政策》。

* 對於排版或非實質性修改，我們將更新頁面頂部之「最後更新（lastUpdated）」日期；
* 對於涉及資料收集方式、使用目的或用戶權利之**重大變更**，我們將於生效前透過網站橫幅公告、系統通知或向閣下註冊電郵發送通知；
* 於修訂生效後繼續使用本項目，即表示閣下同意修訂後之私隱政策。

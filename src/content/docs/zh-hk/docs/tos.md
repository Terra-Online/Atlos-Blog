## 定義與背景

- **本項目**：指由開發者團隊（包括 [OEM-1](https://github.com/orgs/Terra-Online/teams/oem-1) 中的 [cirisus](https://github.com/cirisus)、[mwx0621](https://github.com/mwx0621)、[502y](https://github.com/502y) 等貢獻者）開發及維護之[終末地地圖集（Open Endfield Map）](https://opendfieldmap.org/)網站及相關服務，旨在為用戶提供基於遊戲內數據之地圖瀏覽、點位分析、進度追蹤及雲端同步等功能。
- **服務**：指本項目所提供之地圖展示、路徑規劃、用戶互動、帳戶系統服務，以及數據存取與進度追蹤等功能。
- **用戶**：指存取及／或使用本項目之任何個人或組織。
- **用戶帳戶**：指用戶為使用本項目之進階功能（如雲端數據同步、跨裝置存取等）而註冊並持有之專屬數碼憑證。

> **注意：** 本項目有關私隱數據收集標準、Cookie 使用說明、 API 使用規範及內容來源標註之詳情，請參閱獨立之[私隱政策](../privacy)、[Cookie 政策](../cookie)、[API 使用條款](../api-terms)及[內容來源](../credits)。閣下接受本服務條款，即視為已同時知悉並同意上述附屬政策。

## 數據儲存與私隱保障

數據儲存策略視乎用戶使用狀態而定。

### 未登入情況下之儲存

- 未登入帳戶之用戶，其個人偏好設定、標點進度、篩選狀態等數據，**僅儲存在用戶當前使用之瀏覽器本地（Local Storage／IndexedDB）**。
- 在此模式下，更換裝置、瀏覽器或使用無痕模式將不會繼承相關數據。建議訪客用戶定期使用「匯出／匯入數據（Export／Import Data）」功能作本地備份。

### 已登入情況下之儲存

- 已註冊並登入之用戶，其進度數據及應用設定將**加密同步並儲存於本項目之 D1 數據庫**，以實現跨裝置互通。
- 開發團隊承諾，所收集之數據（包括帳戶識別符、加密後之密碼及加密後之遊戲進度數據）**僅用於提供地圖同步服務**。
- 本項目**不會**將閣下之個人資料或操作記錄出售、共享或披露予任何第三方商業機構。

### 數據安全與免責

- 開發團隊已採用行業標準加密措施保障數據庫安全；惟本項目屬非牟利開源項目，未能提供企業級之絕對安全保證。
